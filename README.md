# 🌍 EcoLens AI: Your Sustainability Digital Twin


EcoLens AI is a Generative AI-powered "What-If" engine that simulates your personal environmental impact over a 5-year horizon. Unlike static carbon calculators, EcoLens uses **Behavioral Digital Twin** logic to project future ecological outcomes based on current lifestyle choices.

## 🚀 Key Features
- **Cognitive Digital Twin:** Uses Gemini 2.5 Flash to simulate environmental decay vs. recovery paths.
- **Hyper-Local Context:** Adapts calculations based on regional power grids (from Bengaluru to rural villages).
- **Greenwashing Detector:** Identifies inefficient "eco-friendly" habits and suggests high-impact alternatives.
- **Persistent Memory:** Saves your simulation timeline locally for continuous tracking.

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS Modules
- **AI Brain:** Google Gemini API (GenAI)
- **Deployment:** Vercel / Netlify
